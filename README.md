# Sumário de Pedidos 🏬

No código de hoje, resolvi trazer mais um exercício do meu curso de Java do professor Nelio Alves. 

O projeto consiste em criar uma simulação de um Sumário de Pedidos, onde são coletados os dados do cliente, o status da encomenda e os dados dos produtos. Com esses dados inseridos o programa retorna o Sumário dos Pedidos, com todos os dados do cliente, do produto, o subtotal do pedido, bem como o valor total do pedido. 

O programa aceita uma variação de número de pedidos, no exemplo eu fixei um sumário de 2 pedidos.
